# Hedgies Rarity

generate hedgies.csv:

```shell
python main.py
```

## Rarity algorithm

```shell
Part score = Total occurrences of the part / Num of species of the part / Occurrence of your part

Total score = Sum of part scores
```

The higher the total score, the rare your hedgies is.
